# Starting
## New Post
> bundle exec jekyll post "POST NAME"

## Example Outline
```
layout: post
title: something_blah_blash
date: 2022-08-08 08:08 +0900
tags: [asd, qwe, zxc, aaa, bbb]
categories: [ccc, ddd]
toc: true
```

## Local Serve
> bundle exec jekyll serve
- 127.0.0.1:4000으로 접속

## Deploy
- git push and wait for github actions (takes a few minutes)
